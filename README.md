开发工具：vs2022+sqlserver20199

# 技术框架
## 前端：Bootstrap
## 核心框架：ASP.NET  MVC
## ORM框架：Entity Framework
## 数据库支持：SqlServer

# 项目结构
## --DTO层：业务层DTO对象
## --MODEL层：数据库实体层，使用EF的CodeFirst模式创建数据库
## --DAL层：数据访问层
## --IDAL层：数据访问层接口
## --BLL层：业务逻辑层
## --IBLL层：业务逻辑层接口
## --MVCSite层：前端UI界面
